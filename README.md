# 🎲 高级在线随机数生成器

一个轻量级、无广告、注重隐私的纯前端随机数生成工具。支持自定义范围、小数精度及去重功能。

## ✨ 主要特性

- **完全本地化**：所有计算均在您的浏览器中完成，数据不会上传至服务器，保护隐私。
- **高度可配置**：
  - 支持自定义生成数量（上限 2000 个以保证性能）。
  - 支持任意数值范围（含负数）。
  - 支持选择保留 0-3 位小数。
- **智能逻辑**：
  - **自动纠错**：如果输入的最小值大于最大值，系统会自动交换。
  - **防死循环**：当"不可重复"模式下的请求数量超过数学可能时，会立即提示。
  - **性能保护**：内置数量熔断机制，防止低性能设备卡顿。
- **现代化设计**：响应式布局，完美适配手机与电脑端。

## 🚀 如何使用

1. **输入数量**：填写你想要生成的随机数个数。
2. **设定范围**：输入最小值和最大值。
3. **高级选项**：
   - 选择是否需要小数（默认整数）。
   - 选择是否允许数字重复出现。
4. 点击 **"点击生成"** 即可获取结果。

## 🛠️ 技术栈

- HTML5 / CSS3 (Flexbox布局)
- Vanilla JavaScript (ES6+)
- 部署平台：GitHub Pages

## ⚠️ 注意事项

- **关于性能**：为了防止页面卡死，单次生成数量被限制在 **2000** 个以内。
- **关于唯一性**：如果在极小范围内（例如 1-10）请求生成大量不重复数字（例如 20 个），系统会报错提示，因为这在数学上是不可能的。
